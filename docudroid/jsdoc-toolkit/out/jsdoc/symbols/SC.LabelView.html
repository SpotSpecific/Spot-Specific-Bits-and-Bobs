Field Detail

displayValue
[RO] The value that will actually be displayed. This property is dynamically computed by applying localization, string conversion and other normalization utilities.
escapeHTML
If true, value will be escaped to avoid scripting attacks. This is a default value that can be overridden by the settings on the owner view.
exampleInlineTextFieldView
The exampleInlineTextFieldView property is by default a SC.InlineTextFieldView but it can be set to a customized inline text field view.
fontWeight
Specify the font weight for this. You may pass SC.REGULAR_WEIGHT, or SC.BOLD_WEIGHT.
formatter
Set this to a validator or to a function and the value will be passed through it before being set. This is a default default that can be overidden by the settings in the owner view.
hint
The hint to display if no value is set. Should be used only if isEditable is set to YES.
icon
An optional icon to display to the left of the label. Set this value to either a CSS class name (for spriting) or an image URL.
isEditable
Enables editing using the inline editor.
isEditing
YES if currently editing label view.
isInlineEditorMultiline
If you want the inline editor to be multiline set this property to YES.
localize
If true, then the value will be localized. This is a default default that can be overidden by the settings in the owner view.
textAlign
Set the alignment of the label view.
validator
Validator to use during inline editing. If you have set isEditing to YES, then any validator you set on this property will be used when the label view is put into edit mode.
value
The value of the label. You may also set the value using a content object and a contentValueKey.
